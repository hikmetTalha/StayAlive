STAY ALIVE
Stay Alive is an FPS shooter game. Zombies come in waves from various points, and we protect the sick and immobile man (and ourselves) from these zombies.
Features
Zombies

Wave System

FPS Shooter Game

"Sick Man" to protect

Headshot system

Medkit (Spawns with a certain probability where zombies die)

Controls
Movement: W - A - S - D

Jump: Space

Shoot: Left Mouse Button

How to Play
Zombies spawn randomly from various points on the map. You must use your pistol to protect both yourself and the defenseless "Sick Man".

Zombies attack whoever is closest.

There are 5 waves in the game. The number of zombies and/or their spawn speed increases with each wave.

There is a 10-second break between waves.

If zombies deplete the player's health or the "Sick Man's" health, you lose the game.

If you survive all 5 waves, you win.

Technologies Used
Unity

Visual Studio (C#)

Mixamo

freesound.org

How to Install
Download the game's ZIP file.

Extract the files.

Double-click StayAlive.exe to run the game.

Note: If Windows displays a blue "Windows protected your PC" warning, click "More info" and then click "Run anyway" to launch the game.
